# NJCIA

## Node.js Initial Setup

in VSCode terminal window:
 - npm init -y
 - npm install express
 - npm install pg
 - npm install -D nodemon      Makes it so in Dev mode don't have to
                        restart server for every change to the files.

    npm install cors --save
  
-------------------------------------

Start Server:
    On terminal run:
        npm run start
To Start Server in Dev mode:
        npm run dev

-------------------------------------
